# aubo_i5_full
Aubo i5 robot with Robotiq 2F-85 gripper and pedestal

# Launch the robot with the gripper:
- Create a ros working-space and cakin_make
- Launch the robot in Rviz ```roslaunch aubo_description display.launch```
